# 学生作品展示墙 - 项目总览

## 一、项目简介

**项目名称**：少儿编程学生作品展示系统  
**项目类型**：全栈 Web 应用（展示工具）  
**核心目标**：为少儿编程校区提供数字化作品展示与管理平台  
**目标用户**：校区教师、学生家长、潜在客户、校区管理者

## 三、技术栈总览

```
前端技术栈
├── 框架：React 18
├── 构建：Vite 5
├── 样式：Tailwind CSS 3
├── 状态：Zustand
├── 路由：React Router 6
├── HTTP：Axios
└── 图标：Lucide React

后端技术栈
├── 运行环境：Node.js 20
├── 框架：Express 4
├── 数据库：MongoDB 6
├── ORM：Mongoose 8
├── 认证：JWT
├── 文件上传：Multer
└── 图片处理：Sharp

基础设施
├── 静态部署：Vercel / Netlify
├── 后端部署：Railway / Render
├── 文件存储：阿里云 OSS
└── 数据库：MongoDB Atlas
```
